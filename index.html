🌍 关于 Locus Maps

Locus Maps 是一个基于 Cesium 和 OpenStreetMap 开发的 3D 地球应用。灵感来自于Google地球，名称意为“位置地图”，旨在提供一个轻量、快速、可交互的全球地图浏览体验。

📍 功能包括：3D 地形、卫星影像、全球建筑、光照切换、建筑显示控制、高精度建模显示（自动触发）。

顶部菜单栏按钮点击可切换亮白/昼夜模式，开关3D建筑，开关高精度建模，重置视角及北向上，查看关于信息。

使用人工智能辅助开发。

📁 开放源代码，若你想要代码文件...请访问我的 GitHub 仓库。

🏙️ 高精度建模覆盖区域(已调试完成的)： 丹佛*、华盛顿D.C.、华盛顿州（西雅图附近）、悉尼*、波士顿*、墨尔本*、旧金山*。浏览时自动检测并提示，与3D建筑互斥。建模加载速度较慢，属正常现象。

*：仅市中心小块区域有高精度建模覆盖

以下为使用技巧：

右下角的控件可以手动控制俯仰角度（点击重置至45°）、恢复朝北，指南针显示当前朝向。

*️⃣ 如何快速确定高精度建模具体覆盖范围？搜索城市英文名称，自动定位并手动放大，弹出提示后点击开启高精度模型，观察突然出现色差的地块。使用无路网图层突兀感较弱。

💡tips：Bing的图源在中国大陆路网显示有误，建议在浏览大陆地区时不要使用“Bing Maps Aerial with Labels”图层，使用“Bing Maps Roads"图层时不要开启3D模型，会导致严重错位。浏览非大陆地区均可。

数据来源：Cesium ion · OpenStreetMap · Natural Earth

为节省数据用量，默认底图为Sentinel-2（分辨率较低），适合宏观浏览。如必须查看地面高清细节，请手动切换至 Bing 地图。
